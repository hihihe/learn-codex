# Codex 功能表

## 1. 会话管理

| 命令 | 功能说明 | 适用场景 |
|---|---|---|
| `/new` | 在当前对话中开启一个新聊天 | 想切换到新话题时 |
| `/resume` | 恢复之前保存的聊天 | 继续未完成的任务 |
| `/fork` | 基于当前聊天分叉出一个新会话 | 保留原对话，同时尝试另一种方案 |
| `/compact` | 压缩/总结当前对话，减少上下文占用 | 对话较长，担心上下文超限时 |
| `/rename` | 重命名当前线程 | 方便整理会话列表 |
| `/clear` | 清空终端并开始新聊天 | 想清理界面重新开始时 |

## 2. 模型配置

| 命令 | 功能说明 | 适用场景 |
|---|---|---|
| `/model` | 选择模型和推理强度 | 需要平衡速度、成本、效果时 |
| `/fast` | 开关 Fast 模式，加快推理速度（会增加 plan 使用量） | 更看重响应速度时 |
| `/personality` | 选择 Codex 的沟通风格 | 调整回答风格和协作方式 |
| `/theme` | 选择语法高亮主题 | 调整终端显示风格 |
| `/statusline` | 配置状态栏显示项 | 自定义界面展示信息 |

## 3. 开发辅助

| 命令 | 功能说明 | 适用场景 |
|---|---|---|
| `/init` | 创建 `AGENTS.md` 指令文件 | 给 Codex 配置项目级协作规则 |
| `/plan` | 切换到 Plan 模式 | 需要先规划步骤再执行时 |
| `/collab` | 切换协作模式（实验性） | 调整人与代理的协作方式 |
| `/agent` | 切换当前激活的 agent 线程 | 多个 agent 并行工作时切换查看 |
| `/subagents` | 切换当前激活的 agent 线程 | 在多个 agent 线程之间切换查看 |
| `/skills` | 使用技能增强 Codex 的特定任务表现 | 执行专项任务时 |
| `/review` | 审查当前改动并找问题 | 提交前做代码检查 |
| `/diff` | 查看 Git 差异，包括未跟踪文件 | 检查改了哪些代码 |
| `/mention` | 提及一个文件 | 在对话中明确引用某个文件 |
| `/copy` | 复制最近一次 Codex 输出到剪贴板 | 快速粘贴结果到别处 |
| `/mcp` | 列出已配置的 MCP 工具 | 查看已接入的外部工具能力 |

## 4. 系统控制

| 命令 | 功能说明 | 适用场景 |
|---|---|---|
| `/permissions` | 配置 Codex 允许执行的权限 | 控制文件、命令、系统访问范围 |
| `/sandbox-add-read-dir` | 给沙箱增加某个绝对路径目录的可读权限 | 需要让 Codex 读取额外路径文件时 |
| `/experimental` | 开关实验性功能 | 体验测试中的能力 |
| `/status` | 查看当前会话配置和 token 使用情况 | 排查配置、观察上下文消耗 |
| `/ps` | 查看后台终端列表 | 检查有哪些后台任务在运行 |
| `/stop` | 停止所有后台终端 | 中止后台执行任务 |
| `/feedback` | 向维护者发送日志/反馈 | 提交问题或建议 |
| `/logout` | 退出登录 Codex | 切换账号或结束登录状态 |
| `/exit` | 退出 Codex | 关闭当前程序 |
